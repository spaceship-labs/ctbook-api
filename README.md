# ctbook-api

The API for contratobook.
A [Sails](http://sailsjs.org) api with generated with `sails new --no-frontend`

Contains the following endpoints

- contrato
- empresa
- unidadCompradora
- dependencia

Create, update and destroy operations are forbidden.

## testing

`npm test`
